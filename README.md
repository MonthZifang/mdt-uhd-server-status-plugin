# uhd-server-status-plugin

在玩家界面的 HUD 区域显示服务器状态信息，配置文件位于 `config/mods/config/uhd-server-status-plugin/config.json`。

## 功能

- 在 HUD 中显示服务器状态
- 可用于展示运行状态、在线信息或其他概览数据
- 适合做大厅服或综合信息展示

## 配置位置

```text
config/mods/config/uhd-server-status-plugin/config.json
```

## 适用场景

- 大厅状态看板
- 玩家进入服务器后的信息展示
- 多服务器状态同步展示
